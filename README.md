# pong-plays-music

A game of pong written in pygame that

- plays a tone in a sequence of notes each time ball hits a pattle
- pattles change colors each time the music sequence loops
- difficulty progressively goes up

Player 1 Controls:
Move up: `Q`
Move Down: `A`

Player 2 Controls:
Move up: `P`
Move Down: `L`


# To build
```
$ python main.py
```
